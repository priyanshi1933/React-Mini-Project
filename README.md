
# Admin Panel

A react-based Admin Panel to manage users, products and orders. It is fetching the data from external APIs and display it in clear and attractive UI. It is used to maintain scalability, maintainability and reliability.


## Tech Stack

**Client:** React, TypeScript, HTML, CSS, BootStrap, React Router




## Installation

Install my-project with npm

```bash
  npm install reactproject
  cd reactproject
  npm start
```
    
## Features

- Product Listing Inside Card Layout
- User Display
- Orders Nabagement
- Responsive Dashboard
- API Data Fetching
- Navigation
- Reusable Components
- Clean UI
